强化学习相关的术语和记号

2026/2/2 智能技术 RL AI应用

看懂强化学习基本的相关的术语和记号，足以读懂很多应用型工作。
这些工作往往把智能技术当作一个“黑箱”，核心创新在于问题建模——如何定义状态/动作/奖励、如何设计交互接口。

概念描述：

  • Marcov chain 描述一个自主演变的随机系统。
  • MDP 描述一个 agent 与环境交互的系统（在一个 Marcov chain 上游走）。
  • partially observed MDP 在 MDP 的基础上建模了 agent 无法直接、完全反应世界的局限。

形式化定义对比表：

$\mathcal{S}$ $\mathcal{A}$ $\mathcal{O}$ $\mathcal{T}$ $\mathcal{E}$ $r$
Marcov chain $\mathcal{M}$ ✓ ✓
MDP $\mathcal{M}$ ✓ ✓ ✓ ✓
partially observed MDP $\mathcal{M}$ ✓ ✓ ✓ ✓ ✓ ✓

符号说明：

  • 前三个都是集合。
  • $\mathcal{S}$ 是 states space（状态空间）。
  • $\mathcal{A}$ 是 actions space（动作空间）。
  • $\mathcal{O}$ 是 observations space（观测空间）。
  • $\mathcal{T}$ 是 transition operator（转移算子），可以把 $\mathcal{T}$ 和 $s_t$（and $a_t$ in a MDP）运算来得到 $s_{t+1}$。Marcov chain 可以写成用矩阵乘法形式，但 MDP 的 transition operator 是三维所以写的是求和形式。
  • $r$ 是 reward function，$r: \mathcal{S} \times \mathcal{A} \rightarrow \mathbb{R}$
  • $\mathcal{E}$ 是 emmision probability $p(o_t|s_t)$（发射概率/观测概率）
